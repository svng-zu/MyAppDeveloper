# 화면 기획서

## 1. 화면 목록
1. 로그인 화면 - 사용자 인증 및 앱 진입
2. 메인 화면 - 1:1 런닝 매칭 기능
3. 내 달리기 현황 화면 - 개인 러닝 기록 및 통계
4. 프로필 화면 - 사용자 정보 관리

## 2. 화면별 상세 구성

### 로그인 화면
**목적:** 사용자 인증을 통한 앱 접근 권한 부여
**구성 요소:**
- 앱 로고 및 타이틀
- 이메일 입력 필드
- 비밀번호 입력 필드
- 로그인 버튼
- 회원가입 버튼
- 소셜 로그인 옵션 (카카오, 구글)

**기능:**
- 이메일/비밀번호 로그인
- 소셜 로그인 연동
- 자동 로그인 설정
- 회원가입 페이지 이동

### 메인 화면
**목적:** 1:1 런닝 파트너 매칭 및 러닝 시작
**구성 요소:**
- 상단 프로필 아이콘 및 알림
- 현재 위치 표시
- 매칭 조건 설정 영역
  - 거리 설정 (1km, 3km, 5km, 10km)
  - 페이스 설정 (여유롭게, 보통, 빠르게)
  - 성별 선택 (무관, 남성, 여성)
- 매칭 시작 버튼
- 매칭 대기 상태 표시
- 매칭 성공 시 파트너 정보
- 러닝 시작 버튼

**기능:**
- GPS 기반 위치 확인
- 매칭 조건 설정
- 실시간 매칭 요청
- 매칭 취소
- 파트너와 채팅
- 러닝 시작

### 내 달리기 현황 화면
**목적:** 개인 러닝 기록 조회 및 통계 확인
**구성 요소:**
- 이번 달 총 거리
- 이번 달 총 러닝 횟수
- 평균 페이스
- 최근 러닝 기록 리스트
  - 날짜
  - 거리
  - 시간
  - 페이스
  - 파트너 정보
- 월별/주별 통계 그래프
- 목표 설정 영역

**기능:**
- 러닝 기록 자동 저장
- 통계 데이터 시각화
- 기록 상세보기
- 목표 설정 및 달성률 표시
- 기록 공유

### 프로필 화면
**목적:** 사용자 정보 관리 및 앱 설정
**구성 요소:**
- 프로필 사진
- 닉네임
- 러닝 레벨 (초급/중급/고급)
- 선호 러닝 시간대
- 자기소개
- 설정 메뉴
  - 알림 설정
  - 개인정보 수정
  - 로그아웃
  - 회원탈퇴

**기능:**
- 프로필 정보 수정
- 프로필 사진 변경
- 러닝 선호도 설정
- 알림 on/off
- 계정 관리

## 3. 네비게이션 구조
- 로그인 → 메인 (자동 이동)
- 메인 ↔ 내 달리기 현황 (하단 탭)
- 메인 ↔ 프로필 (하단 탭)
- 메인 → 러닝 중 화면 (매칭 성공 시)
- 프로필 → 로그인 (로그아웃 시)

## 4. 필수 기능

1. **GPS 기반 위치 서비스**: 현재 위치 확인 및 주변 매칭 파트너 검색
2. **실시간 매칭 시스템**: 설정된 조건에 맞는 파트너 자동 매칭
3. **러닝 기록 추적**: GPS를 통한 거리, 시간, 페이스 자동 측정
4. **푸시 알림**: 매칭 성공, 러닝 초대, 기록 달성 알림
5. **채팅 기능**: 매칭된 파트너와의 간단한 소통
6. **안전 기능**: 긴급상황 시 위치 공유 및 신고 기능

**추가 고려사항:**
- 오프라인 상태에서도 러닝 기록 저장 가능
- 배터리 최적화를 위한 백그라운드 동작 관리
- 사용자 안전을 위한 신원 확인 시스템
