# 화면 기획서

## 1. 화면 목록
1. **온보딩/로그인** - 앱 소개 및 소셜 로그인
2. **메인 홈** - 대시보드 및 주요 기능 접근
3. **런닝 기록** - 실시간 런닝 추적 및 기록
4. **1:1 대결** - 상대방과의 런닝 대결 기능
5. **대결 매칭** - 상대방 찾기 및 매칭
6. **대결 결과** - 대결 완료 후 결과 화면
7. **런닝 기록 상세** - 개별 런닝 기록 분석
8. **커뮤니티** - 게시글, 챌린지, 랭킹
9. **프로필** - 개인 정보 및 통계
10. **설정** - 앱 환경설정

## 2. 화면별 상세 구성

### 온보딩/로그인
**목적:** 신규 사용자 가이드 및 계정 연동
**구성 요소:**
- 앱 소개 슬라이드 (3-4개)
- 소셜 로그인 버튼 (Google, Apple, Kakao)
- 이용약관 및 개인정보처리방침 동의
- 위치 권한, 알림 권한 요청

**기능:**
- 소셜 계정 연동
- 필수 권한 획득
- 초기 프로필 설정

### 메인 홈
**목적:** 앱의 핵심 기능 접근 및 현재 상태 확인
**구성 요소:**
- 상단: 프로필 이미지, 닉네임, 레벨
- 오늘의 런닝 요약 (거리, 시간, 칼로리)
- 빠른 시작 버튼 (혼자 런닝, 1:1 대결)
- 진행 중인 대결 상태
- 최근 활동 피드
- 하단 네비게이션 바

**기능:**
- 런닝 모드 선택
- 대결 현황 확인
- 친구 활동 확인

### 런닝 기록
**목적:** 실시간 런닝 추적 및 데이터 수집
**구성 요소:**
- 지도 (현재 위치, 경로 표시)
- 실시간 데이터 (거리, 시간, 속도, 심박수)
- 음성 가이드 설정
- 일시정지/종료 버튼
- 음악 컨트롤
- 안전 모드 토글

**기능:**
- GPS 기반 위치 추적
- 실시간 런닝 데이터 측정
- 음성 코칭 제공
- 긴급상황 알림 기능

### 1:1 대결
**목적:** 실시간 상대방과의 런닝 경쟁
**구성 요소:**
- 대결 상대 정보 (프로필, 레벨)
- 실시간 비교 차트 (거리, 속도)
- 지도 (내 위치, 상대방 위치)
- 응원 메시지 보내기
- 대결 포기 버튼
- 실시간 순위 표시

**기능:**
- 실시간 위치 공유
- 성과 비교 및 시각화
- 응원/도발 메시지 전송
- 안전 거리 유지 알림

### 대결 매칭
**목적:** 적절한 상대방과의 매칭
**구성 요소:**
- 매칭 조건 설정 (거리, 레벨, 지역)
- 추천 상대방 리스트
- 상대방 프로필 (레벨, 기록, 리뷰)
- 대결 신청/수락 버튼
- 매칭 대기 상태 표시

**기능:**
- 실력 기반 매칭 알고리즘
- 위치 기반 상대 추천
- 대결 조건 협의
- 매칭 히스토리 관리

### 대결 결과
**목적:** 대결 완료 후 결과 분석 및 보상
**구성 요소:**
- 승패 결과 표시
- 상세 비교 데이터 (시간, 거리, 속도)
- 경로 지도 비교
- 획득 포인트/배지
- 상대방 평가하기
- SNS 공유 버튼

**기능:**
- 종합 성과 분석
- 보상 지급
- 상대방 매너 평가
- 소셜 공유

### 런닝 기록 상세
**목적:** 개별 런닝 세션의 상세 분석
**구성 요소:**
- 경로 지도
- 상세 통계 (구간별 속도, 심박수)
- 그래프 (속도, 고도, 심박수)
- 사진/메모 추가
- 목표 달성도
- 개선 제안

**기능:**
- 데이터 시각화
- 성과 분석
- 개인 기록 관리
- 목표 설정 및 추적

### 커뮤니티
**목적:** 사용자 간 소통 및 동기부여
**구성 요소:**
- 피드 타임라인
- 챌린지 목록
- 전체/친구 랭킹
- 게시글 작성 버튼
- 좋아요/댓글 기능
- 해시태그 검색

**기능:**
- 게시글 작성/조회
- 챌린지 참여
- 랭킹 시스템
- 소셜 네트워킹

### 프로필
**목적:** 개인 정보 및 런닝 통계 관리
**구성 요소:**
- 프로필 사진 및 기본 정보
- 누적 통계 (총 거리, 시간, 칼로리)
- 레벨 및 배지 컬렉션
- 최근 활동 기록
- 팔로워/팔로잉 관리
- 개인 목표 설정

**기능:**
- 프로필 편집
- 통계 조회
- 친구 관리
- 목표 설정

### 설정
**목적:** 앱 환경설정 및 계정 관리
**구성 요소:**
- 계정 정보
- 알림 설정
- 개인정보 보호 설정
- 단위 설정 (km/mile)
- 음성 가이드 설정
- 고객 지원
- 로그아웃

**기능:**
- 환경설정 관리
- 개인정보 보호
- 고객 지원 연결
- 계정 관리

## 3. 네비게이션 구조
- **하단 탭:** 홈 → 대결 → 커뮤니티 → 프로필
- **홈:** → 런닝 기록 → 런닝 상세
- **대결:** → 매칭 → 1:1 대결 → 결과
- **커뮤니티:** → 게시글 상세 → 프로필
- **프로필:** → 설정 → 런닝 기록 목록

## 4. 필수 기능
1. **실시간 GPS 추적**: 정확한 위치 및 경로 기록
2. **소셜 로그인**: Google, Apple, Kakao 연동
3. **실시간 대결**: WebSocket 기반 실시간 데이터 동기화
4. **매칭 시스템**: 실력/지역 기반 상대방 추천
5. **안전 기능**: 긴급상황 알림, 안전 거리 유지
6. **데이터 분석**: 런닝 성과 분석 및 시각화
7. **커뮤니티**: 게시글, 댓글, 좋아요 시스템
8. **알림 시스템**: 대결 요청, 응원 메시지, 기록 갱신
9. **오프라인 모드**: 네트워크 없이도 기본 기록 가능
10. **데이터 백업**: 클라우드 동기화 및 백업
