# 화면 기획서

## 1. 화면 목록
1. 스플래시/로그인 화면 - 앱 진입 및 사용자 인증
2. 메인 홈 화면 - 대시보드 및 주요 기능 접근
3. 런닝 준비 화면 - 런닝 시작 전 설정
4. 런닝 기록 화면 - 실시간 런닝 추적 및 기록
5. 실시간 대결 화면 - 주변 사용자와 1:1 대결
6. 런닝 결과 화면 - 운동 완료 후 결과 요약
7. 기록 관리 화면 - 과거 런닝 기록 조회
8. 프로필 화면 - 사용자 정보 및 설정
9. 랭킹 화면 - 전체/지역 랭킹 조회
10. 친구/커뮤니티 화면 - 소셜 기능

## 2. 화면별 상세 구성

### 스플래시/로그인 화면
**목적:** 사용자 인증 및 앱 진입
**구성 요소:**
- 앱 로고 및 브랜딩
- 소셜 로그인 버튼 (구글, 카카오, 애플)
- 이용약관 및 개인정보처리방침 링크
- 게스트 모드 진입 버튼

**기능:**
- 소셜 계정 연동 로그인
- 자동 로그인 (토큰 저장)
- 위치 권한 요청
- 게스트 모드 제공 (제한된 기능)

### 메인 홈 화면
**목적:** 앱의 중심 허브 역할 및 주요 정보 표시
**구성 요소:**
- 상단 프로필 아이콘 및 알림 아이콘
- 오늘의 런닝 요약 카드 (거리, 시간, 칼로리)
- 주간/월간 통계 그래프
- "런닝 시작" 메인 버튼
- "실시간 대결" 버튼
- 최근 대결 결과 미리보기
- 날씨 정보 위젯
- 추천 코스 카드

**기능:**
- 런닝 시작 화면으로 이동
- 실시간 대결 모드 진입
- 통계 데이터 시각화
- 날씨 API 연동
- 푸시 알림 관리

### 런닝 준비 화면
**목적:** 런닝 시작 전 필요한 설정
**구성 요소:**
- 목표 설정 섹션 (거리/시간/칼로리)
- 런닝 모드 선택 (일반/대결 대기)
- 음악 연동 버튼
- GPS 신호 상태 표시
- 런닝 코스 미리보기 지도
- "시작하기" 버튼

**기능:**
- 목표값 설정 및 저장
- Spotify/Apple Music 연동
- GPS 정확도 확인
- 대결 모드 활성화
- 런닝 경로 추천

### 런닝 기록 화면
**목적:** 실시간 런닝 데이터 추적 및 표시
**구성 요소:**
- 실시간 지도 (현재 위치, 경로)
- 주요 지표 (속도, 거리, 시간, 심박수)
- 랩타임 기록 버튼
- 일시정지/재개/종료 버튼
- 음성 코칭 on/off 스위치
- 대결 상대 정보 (대결 모드 시)

**기능:**
- GPS 기반 실시간 위치 추적
- 운동 데이터 실시간 계산
- 음성 안내 (구간별 기록 안내)
- 웨어러블 기기 연동 (심박수)
- 실시간 대결 데이터 동기화

### 실시간 대결 화면
**목적:** 주변 사용자와 실시간 경쟁
**구성 요소:**
- 대결 상대 프로필 정보
- 실시간 순위 표시
- 나와 상대방 진행 상황 비교 그래프
- 대결 조건 표시 (거리/시간 기준)
- 응원 메시지 전송 버튼
- 포기/계속하기 버튼
- 실시간 채팅 (선택사항)

**기능:**
- 위치 기반 상대방 매칭
- 실시간 데이터 동기화
- 대결 결과 실시간 계산
- 채팅 및 이모티콘 전송
- 대결 포기 시 패널티 적용

### 런닝 결과 화면
**목적:** 운동 완료 후 상세 결과 및 성과 표시
**구성 요소:**
- 전체 경로 지도
- 주요 성과 요약 (거리, 시간, 속도, 칼로리)
- 구간별 상세 분석 (랩타임, 페이스)
- 대결 결과 (승/패, 상대 정보)
- 개인 기록 갱신 알림
- SNS 공유 버튼
- 저장하기 버튼

**기능:**
- 운동 데이터 저장
- 개인 기록 비교 및 갱신
- 소셜 미디어 공유
- 대결 승부 기록 업데이트
- 칼로리 소모량 계산

### 기록 관리 화면
**목적:** 과거 런닝 기록 조회 및 관리
**구성 요소:**
- 날짜별 기록 리스트
- 필터링 옵션 (기간, 거리, 대결 여부)
- 통계 차트 (주간/월간/년간)
- 개인 기록 현황
- 대결 승률 통계
- 즐겨찾기 코스
- 데이터 내보내기 버튼

**기능:**
- 기록 검색 및 필터링
- 데이터 시각화
- 즐겨찾기 코스 관리
- CSV/PDF 내보내기
- 기록 삭제 기능

### 프로필 화면
**목적:** 사용자 정보 관리 및 앱 설정
**구성 요소:**
- 프로필 사진 및 닉네임
- 총 런닝 통계 (총 거리, 시간, 횟수)
- 뱃지/업적 컬렉션
- 대결 전적 (승/무/패)
- 개인정보 수정 버튼
- 앱 설정 (알림, 단위, 음성안내)
- 로그아웃 버튼

**기능:**
- 프로필 정보 수정
- 업적 시스템 관리
- 알림 설정 관리
- 단위 설정 (km/mile)
- 개인정보 보호 설정

### 랭킹 화면
**목적:** 사용자들 간의 순위 및 경쟁 현황 표시
**구성 요소:**
- 탭 메뉴 (전체/지역/친구)
- 기간별 랭킹 (일간/주간/월간)
- 카테고리별 랭킹 (거리/시간/대결승률)
- 내 순위 하이라이트
- 상위 랭커 프로필
- 시상대 UI (1,2,3위)

**기능:**
- 실시간 랭킹 업데이트
- 지역별 필터링 (GPS 기반)
- 랭킹 카테고리 전환
- 사용자 프로필 조회
- 랭킹 공유 기능

### 친구/커뮤니티 화면
**목적:** 소셜 기능 및 커뮤니티 활동
**구성 요소:**
- 친구 목록 및 상태
- 친구 추가 버튼 (ID/연락처)
- 최근 활동 피드
- 커뮤니티 게시판
- 그룹 대결 참여 버튼
- 메시지 기능

**기능:**
- 친구 검색 및 추가
- 활동 피드 실시간 업데이트
- 댓글 및 좋아요 기능
- 그룹 대결 생성/참여
- 1:1 메시지 기능

## 3. 네비게이션 구조
- 메인 홈 → 런닝 준비 → 런닝 기록 → 런닝 결과
- 메인 홈 → 실시간 대결 → 대결 진행 → 대결 결과
- 하단 탭: 홈/기록/랭킹/커뮤니티/프로필
- 모든 화면에서 프로필 접근 가능
- 런닝 중에는 최소한의 네비게이션만 제공

## 4. 필수 기능
1. **GPS 추적**: 정확한 위치 기반 거리 및 경로 측정
2. **실시간 매칭**: 주변 사용자와 즉시 대결 연결
3. **데이터 동기화**: 대결 중 실시간 진행상황 공유
4. **웨어러블 연동**: Apple Watch, 갤럭시 워치 등 연동
5. **음성 안내**: 구간별 기록 및 대결 상황 음성 안내
6. **백그라운드 실행**: 앱을 닫아도 기록 지속
7. **오프라인 모드**: 인터넷 연결 없이도 기본 기록 기능
8. **데이터 보안**: 개인 위치 정보 암호화 및 보안
9. **푸시 알림**: 대결 요청, 기록 갱신 등 실시간 알림
10. **소셜 공유**: 성과를 SNS에 공유하는 기능
