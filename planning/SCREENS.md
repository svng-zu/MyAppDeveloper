# 화면 기획서

## 1. 화면 목록
1. **스플래시 화면** - 앱 로딩 및 브랜딩
2. **로그인/회원가입 화면** - 사용자 인증
3. **메인 대시보드** - 런닝 기록 요약 및 주요 기능 접근
4. **런닝 기록 화면** - GPS 기반 실시간 런닝 추적
5. **기록 상세 화면** - 개별 런닝 기록 상세 정보
6. **대결 목록 화면** - 1:1 대결 현황 및 참여
7. **대결 생성 화면** - 새로운 대결 만들기
8. **대결 상세 화면** - 대결 진행 상황 및 결과
9. **프로필 화면** - 사용자 정보 및 통계
10. **설정 화면** - 앱 설정 및 계정 관리
11. **친구 목록 화면** - 친구 관리 및 검색

## 2. 화면별 상세 구성

### 스플래시 화면
**목적:** 앱 브랜딩 및 초기 로딩
**구성 요소:**
- 앱 로고
- 앱 이름
- 로딩 인디케이터
- 버전 정보

**기능:**
- 자동 로그인 체크
- 앱 권한 확인 (위치, 카메라 등)
- 메인 화면 또는 로그인 화면으로 자동 이동

### 로그인/회원가입 화면
**목적:** 사용자 인증 및 계정 생성
**구성 요소:**
- 이메일/아이디 입력 필드
- 비밀번호 입력 필드
- 로그인 버튼
- 회원가입 버튼
- 소셜 로그인 버튼 (구글, 카카오)
- 비밀번호 찾기 링크

**기능:**
- 입력 유효성 검증
- 자동 로그인 옵션
- 소셜 로그인 연동
- 회원가입 시 닉네임, 프로필 사진 설정

### 메인 대시보드
**목적:** 앱의 중심 허브 역할
**구성 요소:**
- 상단 프로필 이미지 및 인사말
- 오늘의 런닝 요약 카드
- 이번 주 통계 (거리, 시간, 칼로리)
- 빠른 런닝 시작 버튼
- 진행 중인 대결 알림
- 최근 대결 결과 미리보기
- 하단 네비게이션 바 (홈, 기록, 대결, 프로필)

**기능:**
- 원터치 런닝 시작
- 대결 현황 실시간 업데이트
- 주간/월간 통계 전환
- 알림 확인 및 관리

### 런닝 기록 화면
**목적:** 실시간 런닝 추적 및 기록
**구성 요소:**
- 지도 뷰 (GPS 경로 표시)
- 실시간 통계 패널 (거리, 시간, 페이스, 칼로리)
- 시작/일시정지/정지 버튼
- 음성 안내 토글
- 음악 컨트롤러
- 현재 위치 버튼
- 목표 설정 영역

**기능:**
- GPS 기반 거리 및 경로 추적
- 실시간 통계 계산 및 표시
- 음성 안내 (거리별, 시간별)
- 백그라운드 실행 지원
- 런닝 데이터 자동 저장
- 목표 달성 알림

### 기록 상세 화면
**목적:** 개별 런닝 기록의 상세 분석
**구성 요소:**
- 경로 지도
- 상세 통계 (거리, 시간, 평균/최고 속도, 칼로리, 고도)
- 구간별 페이스 그래프
- 사진/메모 영역
- 공유 버튼
- 삭제/수정 옵션

**기능:**
- 경로 재생 애니메이션
- 구간별 분석 데이터 제공
- 사진 첨부 및 메모 작성
- SNS 공유 기능
- 기록 편집/삭제

### 대결 목록 화면
**목적:** 진행 중이거나 참여 가능한 대결 관리
**구성 요소:**
- 탭 메뉴 (진행중, 완료, 초대받은)
- 대결 카드 리스트
  - 상대방 프로필 사진
  - 대결 제목 및 조건
  - 진행 상황 바
  - D-day 표시
- 새 대결 생성 FAB 버튼
- 검색 기능

**기능:**
- 대결 상태별 필터링
- 실시간 진행 상황 업데이트
- 대결 참여/거절
- 친구 초대 기능

### 대결 생성 화면
**목적:** 새로운 1:1 대결 설정
**구성 요소:**
- 상대방 선택 (친구 목록)
- 대결 조건 설정
  - 목표 (거리, 시간, 빈도)
  - 기간 (일주일, 한달 등)
  - 승부 조건
- 제목 입력
- 내기 설정 (선택사항)
- 생성 버튼

**기능:**
- 친구 검색 및 선택
- 다양한 대결 조건 설정
- 초대 메시지 발송
- 대결 유효성 검증

### 대결 상세 화면
**목적:** 특정 대결의 진행 상황 및 결과 확인
**구성 요소:**
- 상대방 정보
- 대결 조건 요약
- 진행 상황 비교 차트
- 일별 기록 타임라인
- 채팅 영역 (간단한 응원 메시지)
- 포기 버튼

**기능:**
- 실시간 진행률 비교
- 상대방과 간단한 소통
- 대결 포기 기능
- 결과 공유

### 프로필 화면
**목적:** 사용자 정보 및 런닝 통계 관리
**구성 요소:**
- 프로필 사진 및 기본 정보
- 레벨 및 뱃지 시스템
- 누적 통계 (총 거리, 시간, 대결 승률)
- 최근 활동 그래프
- 달성 뱃지 갤러리
- 개인 기록 (최장 거리, 최고 속도 등)

**기능:**
- 프로필 정보 수정
- 통계 기간 선택
- 뱃지 및 업적 시스템
- 개인 기록 갱신 알림

### 설정 화면
**목적:** 앱 환경 설정 및 계정 관리
**구성 요소:**
- 계정 설정 (정보 수정, 비밀번호 변경)
- 알림 설정
- 단위 설정 (km/mile, kg/lb)
- 음성 안내 설정
- 개인정보 처리방침
- 로그아웃/회원탈퇴

**기능:**
- 개인정보 수정
- 푸시 알림 제어
- 측정 단위 변경
- 데이터 백업/복원

### 친구 목록 화면
**목적:** 친구 관리 및 소셜 기능
**구성 요소:**
- 친구 검색 바
- 친구 목록 (프로필, 최근 활동)
- 친구 요청 탭
- 추천 친구 영역
- 초대 버튼

**기능:**
- 친구 검색 및 추가
- 친구 요청 승인/거절
- 친구 활동 모니터링
- 앱 초대 기능

## 3. 네비게이션 구조
```
스플래시 → 로그인/회원가입 → 메인 대시보드
            ↓
메인 대시보드 ↔ 런닝 기록 화면
    ↓           ↓
[하단 네비게이션]  기록 상세 화면
- 홈 (메인 대시보드)
- 기록 (기록 목록)
- 대결 (대결 목록) → 대결 생성/상세
- 프로필 → 설정, 친구 목록
```

## 4. 필수 기능
1. **GPS 추적**: 정확한 위치 기반 거리 및 경로 측정
2. **실시간 동기화**: Firebase 기반 대결 데이터 실시간 업데이트
3. **백그라운드 실행**: 런닝 중 앱이 백그라운드에서도 추적 지속
4. **오프라인 지원**: 네트워크 연결 없이도 기본 기록 기능 동작
5. **데이터 암호화**: 사용자 개인정보 및 운동 데이터 보안
6. **소셜 로그인**: 구글, 카카오 등 간편 로그인 지원
7. **푸시 알림**: 대결 상황, 목표 달성 등 실시간 알림
8. **음성 안내**: TTS 기반 런닝 중 음성 피드백
9. **데이터 백업**: 클라우드 기반 운동 기록 백업/복원
10. **친구 시스템**: 친구 추가, 대결 초대, 활동 공유
