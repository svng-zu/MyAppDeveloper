# PRD: 기획 다시 진행

안녕하세요! 새로운 앱 기획을 시작하겠습니다.

어떤 앱을 기획하고 싶으신가요? 다음과 같은 정보를 알려주시면 더 상세한 PRD를 작성해드릴 수 있습니다:

1. **앱의 기본 아이디어나 컨셉**
2. **해결하고자 하는 문제나 니즈**
3. **대략적인 타겟 사용자층**
4. **참고하고 있는 유사 서비스 (있다면)**

예를 들어:
- "운동 습관 형성을 돕는 앱"
- "반려동물 관리 앱"
- "지역 기반 중고거래 앱"
- "스터디 그룹 매칭 앱"

등등 어떤 분야든 상관없습니다. 아이디어를 말씀해주세요!
